Use one of the following variables for your templates:

{synonym,word} - Synonym
  * word: Common English noun, adjective or adverb

{theme} - Current theme's name
{color} - One of the predominant colors
{colour} - Alias of {color}
{emotion} - Positive emotion. The last invoked {color} will be used for reference.
{keyword} - Random keyword from keywords.txt
{inspiration} - Inspiration word from the user-defined list
{alt,list} - Random alternative
  * list: Comma-separated list of alternatives
