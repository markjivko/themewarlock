Installation

Step One - Install the theme

  1. Locate the {project.destDir}.zip theme archive
  2. Navigate to the Admin Dashboard on your WordPress website
  3. Go to Appearance > Themes > Add New and click on Upload Theme
  4. Choose the {project.destDir}.zip archive and click on Install
  5. After the theme is installed, Activate it from Appearance > Themes > {utils.common.themeName}

Step Two - Activate the plugins

This theme requires a number of plugins in order to work properly.

  1. Navigate to the Admin Dashboard on your WordPress website
  2. Click on Begin activating plugins from the warning notification
  3. Check all the plugins
  4. Select Activate from the Bulk Actions dropdown
  5. Click on Apply

Step Three - Install the Demo Content

We have prepared multiple snapshots - Demo Content packages - so that you can jump into editing your new website without the need to manually add content elements.

  1. Locate the desired #.zip Demo Content archive
  2. Navigate to the Admin Dashboard on your WordPress website
  3. Go to Appearance > Demo Content and click on the + button
  4. Upload the desired snapshot, for example 1.zip
  5. Click on INSTALL

Widget Blocks

Use editable widget blocks in all widget areas

Create your own beautiful widgets using any content editor you prefer, including WPBakery Page Builder.

Each "widget block" is a custom post type. Edit your own widget blocks by visiting Widget Blocks.

You can then load your custom "widget blocks" in any Widget Area, by visiting Appearance > Widgets and adding the {config.authorName}: Widget Block widget.


Important!

In order to use WPBakery Page Builder to edit your custom widget blocks follow these steps:

  1. Visit WPBakery Page Builder > Role Manager
  2. Click on the Post types drop-down
  3. Select Custom
  4. Check st_widget_block
