# What is it?

This is an Add-On for WPBakery Page Builder that displays a small slider.

Each slide features an image, a title and a short description.

This is useful for displaying testimonials, features etc.

# How to customize it?

## {addon.title}

> {addon.description}

Edit any page using WPBakery Page Builder and add the **{project.destProjectName} - {addon.title}** block.
